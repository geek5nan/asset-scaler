# 快速启动指南

## 1. 安装依赖

```bash
cd web-app
npm install
```

## 2. 启动开发服务器

```bash
npm run dev
```

浏览器会自动打开 http://localhost:5173

## 3. 使用应用

1. **上传图片**：拖拽 PNG/JPG 图片到上传区域，或点击选择文件
2. **配置选项**（左侧侧边栏）：
   - 选择输入图片倍数（1x/2x/3x/4x）
   - 调整 WebP 质量（滑块）
   - 选择是否生成 drawable/ 目录
3. **开始转换**：点击底部"开始转换"按钮
4. **下载结果**：转换完成后，点击"下载 ZIP 包"

## 4. 构建生产版本

```bash
npm run build
```

构建后的文件在 `dist` 目录，可以直接部署到：
- GitHub Pages
- Vercel
- Netlify
- 任何静态托管服务

## 注意事项

- 所有处理都在浏览器本地完成，不会上传到服务器
- 配置会自动保存到浏览器的 localStorage
- 支持批量上传和处理多个图片文件
- 生成的 ZIP 包包含完整的 drawable 文件夹结构
